# Project Name

Call graph
## Features

1. obtains access token
1. calls graph
1. displays users

## Getting Started

### Prerequisites

Python 3.7+

### Installation
```
git clone
git submodule init
pip install -r requirements.txt

```

### Quickstart

```
    export FLASK_APP=call_ms_graph.py
    export FLASK_ENV=development
    export FLASK_DEBUG=1
    export FLASK_RUN_CERT=adhoc
    flask run
```
